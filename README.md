# ⏰ Tkinter Alarm Clock

A sleek, dark-mode digital alarm clock built using Python and Tkinter for my **Pinnacle Python Internship (2026)**.

## 🎨 Features
* **Live Digital Display:** Displays a real-time, high-contrast clock.
* **Catppuccin Theme:** Features a beautiful, developer-friendly dark palette (`#1e1e2e`).
* **Precise Alarms:** Set custom target hours, minutes, and seconds using straightforward dropdown controls.
* **Native Alerts:** Triggers a system bell and a clean visual pop-up notice once the countdown target triggers.

## 🚀 How to Run
1. Make sure you have Python installed.
2. Clone this repository or download the source script.
3. Run the application from your terminal:
   ```bash
   python alarm_clock.py
